RPS 1.0

RPS (Resistant Procrustes Software) is a compact and integrated framework to perform landmark-based resistant shape analysis of 2D and 2D datasets.

RPS main functionalities are a set of resistant tools:
a resistant Procrustes superposition,
a resistant distance computation
a resistant MDS.

RPS is an open source software with a well-designed user interface. It has quality and versatile graphics, offering the user a friendly and customizable working environment.

RPS overall performance is reasonable fast and efficient, and because of its modular design new resistant functionalities could be incorporated in the near future.
